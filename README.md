# Curriculum PDF Generator

Python script to generate one-page PDF resumes in dark mode.

## 🚀 How to Run
pip install -r requirements.txt
python src/generator.py

## 📂 Project Structure
- `src/` → main Python script
- `examples/` → sample resume text
- `requirements.txt` → dependencies
- `README.md` → documentation
